# Project ORION

Project ORION is an AI-powered YouTube automation platform.

## Goal

Automatically:

- Collect news
- Rank stories
- Research facts
- Generate scripts
- Generate voice
- Create videos
- Generate thumbnails
- Upload to YouTube

## Current Progress

- ✅ RSS Reader
- ✅ SQLite Database
- ✅ Story Ranking
- ✅ Story Selection
- ✅ Workflow Controller (orion.py)
